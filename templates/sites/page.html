Sites

Add this code to your apache .htaccess to redirect GoogleBot to Seo Server. More extensive instructions coming soon. Post on the Google Group for more help for now.

Enable Seo Server Example
RewriteEngine on
RewriteCond %{QUERY_STRING} ^_escaped_fragment_=(.*)$
RewriteRule (.*) http://seo.apiengine.io:4000/seo/%1? [P]


Enable Sitemap Example - replace `10` with your site id found in sitemap link below
RewriteCond $1 ^sitemap.xml
RewriteRule ^(.*)$ http://seo.apiengine.io:4000/10/sitemap.xml [P]
